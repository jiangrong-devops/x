Antd Next X 是一款 AI 应用复合工具集，融合了 UI 组件库、流式 Markdown 渲染引擎和 AI SDK，为开发者提供构建下一代 AI 驱动应用的完整工具链。

**`@antdv-next/x` - 智能界面构建框架**

基于 Antdv Next 设计体系的 Vue UI 库、专为 AI 驱动界面设计，开箱即用的智能对话组件、无缝集成 API 服务，快速搭建智能应用界面，查看详情请点击 [这里](/components)。

**`@antdv-next/x-markdown` - 高性能流式渲染引擎**

专为流式内容优化的 Markdown 渲染解决方案、强大的扩展能力，支持公式、代码高亮、mermaid 图表等极致性能表现，确保流畅的内容展示体验。查看详情请点击 [这里](/markdown)。

**`@antdv-next/x-sdk` - AI 对话数据流管理**

提供完整的工具 API 集合、开箱即用的 AI 对话应用数据流管理、简化开发流程，提升开发效率。查看详情请点击 [这里](/sdk)。

**`@antdv-next/x-skill` - 智能技能库**

是专为 Antd Next X 打造的智能技能库，提供了一系列精心设计的 Agent 技能。这些技能能够显著提升开发效率，帮助您快速构建高质量的 AI 对话应用，并有效解决开发过程中遇到的各种问题，详情点击 [这里](/skill)。

## 如何贡献

在任何形式的参与前，请先阅读 [贡献者文档](https://github.com/antdv-next/antdv-next/blob/main/CODE_OF_CONDUCT.md)。如果你希望参与贡献，欢迎提交 [Pull Request](https://github.com/antdv-next/x/pulls)，或给我们 [报告 Bug](https://github.com/antdv-next/x/issues)。

> 强烈推荐阅读 [《提问的智慧》](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way)、[《如何向开源社区提问题》](https://github.com/seajs/seajs/issues/545) 和 [《如何有效地报告 Bug》](http://www.chiark.greenend.org.uk/%7Esgtatham/bugs-cn.html)、[《如何向开源项目提交无法解答的问题》](https://zhuanlan.zhihu.com/p/25795393)，更好的问题更容易获得帮助。
