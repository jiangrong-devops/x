## ✨ 特性

- 🌈 提炼自 Ant Design X 的交互语言和视觉风格。
- 📦 开箱即用的高质量 Vue3 AI 组件。
- 📝 强大的 Markdown 渲染能力，支持流式渲染、公式、代码高亮、Mermaid 等。
- 🛡 使用 TypeScript 开发，提供完整的类型定义文件。
- ⚙️ 与 Antdv Next 一致的开发体验。
- 🌍 数十个国际化语言支持。
- 🎨 基于 CSS-in-JS 的主题定制能力。
