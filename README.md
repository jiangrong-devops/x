## ✨ Features

- 🌈 Extracted from Ant Design X interactive language and visual style.
- 📦 Out-of-the-box high-quality Vue3 AI components.
- 📝 Powerful Markdown rendering capabilities, supporting streaming rendering, formulas, code highlighting, Mermaid, etc.
- 🛡 Developed with TypeScript, providing complete type definition files.
- ⚙️ Consistent development experience with Antdv Next.
- 🌍 Support for dozens of international languages.
- 🎨 Theme customization capabilities based on CSS-in-JS.
